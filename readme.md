SVCommuniter Chrome Extension
-----------------------------

It modifies and adds functionality to the online community [SVCommunity.org][1]

![popup][2]

![options][3]

*Features*

* show recent posts
* code highlighting
* new ways of searching 
* keyboard navigation
* blacklist

*Keyboard shortcuts*

* `j` siguiente mensaje
* `k` mensaje anterior
* `o` abrir post
* `u` mensajes no leidos
* `p` mensajes privados
* `h` foro principal
* `s` busqueda
